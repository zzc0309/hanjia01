
##### 简要描述

- 用户登录接口
##### 请求URL
- ` http://8.129.112.196:8081/login `
  
##### 请求方式
- POST 

##### 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|username |是  |string |用户名   |
|password |是  |string | 密码    |

##### 返回示例 

``` 
{
    "message": "登录失败",
    "status": 404
}
```

##### 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|message |string   |描述信息  |
|status|int|200,400:登录成功; 404:登录失败|

##### 备注 

- 需使用表单请求,返回数据为JSON格式
